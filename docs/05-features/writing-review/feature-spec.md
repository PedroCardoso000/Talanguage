# Feature Spec — Writing Review

## Objetivo
Permitir que o usuário pratique escrita em inglês ou espanhol, envie uma resposta curta e receba feedback simples com pontos de melhoria.

## Contexto atual
A tela de Escrever/Revisar já existe mockada. A primeira versão deve transformar o fluxo visual em funcionalidade conectada ao backend.

## Escopo da primeira versão
Inclui:
- escolher idioma;
- escolher nível;
- escolher desafio de escrita;
- escrever resposta;
- enviar para revisão;
- receber feedback simples;
- visualizar pontos fortes e melhorias;
- salvar atividade no progresso.

## Fora de escopo
- correção gramatical avançada em tempo real;
- IA real obrigatória;
- redações longas;
- avaliação certificadora;
- plágio;
- revisão humana.

## Critério de pronto
- desafio carregado do backend;
- envio de resposta funcional;
- feedback retornado pelo backend;
- atividade registrada para progresso;
- tela mockada desmockada sem recriar layout.
