# Talanguage — Implementation Prompts

Esta pasta contém prompts sequenciais para orientar o Codex na implementação progressiva do Talanguage.

## Como usar

1. Abra o próximo arquivo da sequência em `docs/11-prompts/implementation/`.
2. Envie o conteúdo para o Codex.
3. Aguarde a execução.
4. Rode validações quando o prompt solicitar.
5. Atualize o status em `docs/11-prompts/implementation-status.md`.
6. Avance para o próximo prompt.

## Regra principal

Não pule etapas. Cada prompt existe para reduzir risco, token e delírio arquitetural.

## Comando operacional

Quando quiser continuar, use o próximo prompt numerado.
