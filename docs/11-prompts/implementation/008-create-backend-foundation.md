# Prompt 008 — Criar Fundação Backend

## Objetivo

Criar a fundação do backend conforme stack autorizada, sem implementar features completas ainda.

## Atenção

Antes de executar, confirme que `docs/01-ai-contract/allowed-stack.md` define oficialmente a stack do backend.

## Tarefa

Criar a estrutura inicial do backend de acordo com a stack autorizada.

## Regras

- Não implementar feature de negócio ainda.
- Não criar banco real se database-rules não estiverem definidos para isso.
- Não adicionar bibliotecas fora da stack autorizada.
- Não criar microserviços.
- Não criar IA real.
- Preparar camadas de domínio, aplicação, infraestrutura e API.

## Entrega esperada

Informe:

1. estrutura criada;
2. comandos necessários para rodar;
3. validações executadas;
4. decisões assumidas;
5. pendências antes da Auth real.
