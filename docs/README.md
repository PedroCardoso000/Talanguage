# Talanguage Docs — Phase 7

This package contains feature specifications for the first version of Talanguage.

Context:
- The frontend already exists with mocked UI and mocked data.
- The backend does not exist yet.
- These specs guide the progressive transition from mocked UI to functional full-stack features.

Included features:
- Auth
- Dashboard
- Speaking Practice
- Writing Review
- Flashcards
- Community
- Progress

Each feature includes:
- feature-spec.md
- domain-spec.md
- api-spec.md
- backend-spec.md
- frontend-spec.md
- test-spec.md
